## 구현 시사점
- AE 렌더 루프에서 모델 로드/파일 I/O 금지 규칙은 유지해야 함.
- ORT는 세션 1회 초기화 + 프레임별 재사용 구조가 필수.
- GPU 백엔드는 OS별 지원이 다르므로 EP fallback 순서를 명시해야 함.
- Depth Anything 계열은 버전/배포 형태가 변동 가능하므로, 모델 카탈로그를 코드 하드코딩보다 manifest 기반으로 전환하는 것이 안전함.
